# Exploratory Data Analysis and Visualization of Obesity Dataset using Python and Google Colab

## Project Overview
This project focuses on Exploratory Data Analysis (EDA) of an obesity dataset to identify factors influencing obesity levels. The analysis examines demographic, lifestyle, eating habits, physical activity, and transportation-related attributes to uncover meaningful patterns and trends.

### Columns Included : 
- Gender
- Age
- Height
- Weight
- Family History
- Frequent Consumption of High-Calorie Food (FAVC)
- Vegetable Consumption (FCVC)
- Number of Main Meals (NCP)
- Food Consumption Between Meals (CAEC)
- Smoking
- Water Consumption (CH2O)
- Calorie Monitoring (SCC)
- Physical Activity Frequency (FAF)
- Time Using Technology Devices (TUE)
- Alcohol Consumption (CALC)
- Transportation Method (MTRANS)
- Obesity Level  

## Objectives
- Perform data cleaning and preprocessing.
- Explore obesity trends across different demographics.
- Analyze the impact of eating habits and lifestyle choices.
- Visualize relationships between obesity levels and various factors.
- Generate actionable insights using data visualization.

## Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Data Cleaning
- Checked dataset structure and data types.
- Verified missing values.
- Examined feature distributions.
- Prepared data for visualization and analysis.

## Key Insights
- Family history plays a significant role in obesity levels.
- Physical activity is associated with lower obesity rates.
- High-calorie food consumption contributes to higher obesity levels.
- Transportation and lifestyle habits influence obesity patterns.
- Eating behavior and daily routines show strong correlations with obesity categories.
